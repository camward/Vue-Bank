# Vue Bank

Склонировать проект: `https://github.com/camward/Vue-Bank.git`

Перейти в папку проекта и установить зависимости: `npm i` или `yarn`

Запустить проект командой `npm run serve` и перейти по адресу `http://localhost:8080/`
